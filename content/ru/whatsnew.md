
## Что нового (Версия 1.0.2)
<br>

#### Полная пересборка сервиса плейлистов:
- Поддержка больших плейлистов (Для 100к+ треков: Первое добавление - зависит от скорости диска и обьёма метаданных. Сохранение/Загрузка ~1.5сек.)
- Ассинхроное чтение метаданных, треки в плейлисте появляются практически сразу, нет необходимости ждать пока загрузится полностью.
- Чтение метаданных для сетевых источников
- Импорт/Экспорт всех плейлистов в свой бинарный формат
- Открытие/сохранение плейлистов форматов .m3u, .m3u8, .pls, .xspf, .cue
- Быстрая сортировка/поиск (100к+ треков, ~0.8...1сек)
- Отображение плейлистов переведено на модель/отображение, минимизация потребления памяти отображением
- Сохранения путей добавления файлов по каждому плейлисту
- Автоименование плейлистов по метаданным/именам/родительской папке (выбирается автоматически), больше никаких "Playlist X".
- Расширеный автоматический поиск обложек
<br>

#### Оптимизация сервиса закладок, меньше памяти, выше скорость (1к+ закладок)
- Добалена обработка невалидных закладок при отсутствии плейлиста/трека.
<br>

#### Полная пересборка аудиодвижка (FFMpeg + PortAudio + VST2)
- Поддержка воспроизведения ещё большего множества форматов
- Поддержка сетевых источников (перемотка работает только на допускающих это источниках, например для интернет-радио это невозможно)
- добавлен нативный модуль измерения сигнала, больше никаких внешних вычеслений.
- Значительная оптимизация загрузки процессора для всего процесса воспроизведения
- Добавлена возможность сохранять пресеты для VST плагинов
- Аудиопоток теперь исполняется без GIL, практически полный натив на C (Cython)
- Все аудио-процессы исполняются в 32 bit float
- Добавлен ресемплинг (с возможностью отключить)
<br>

#### Добавлен Веб интерфейс для управление плеером по сети
- Встроенный клиент
- Поддержка внешних клиентов (WebSockets, API доступно [Здесь](web-interface/web-api.ru.md))
<br>

#### Добавлен режим работы из трея
- Доступен интерфейс мини-плеера
- Управление из контекстного меню трея
<br>

#### Добавлено меню в контекстное меню проводника для папок/файлов
- Воспроизведение (плейлист "по умолчанию"), Добавление в текущий/новый плейлист
<br>

#### Добавлена онлайн библиотека СВиД
- Добавление в текущий, или новый плейлист любой книги (как онлайн источника)
- Возможность скачать книгу и автоматически добавить в плейлист
<br>
