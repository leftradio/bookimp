
## Технические характеристики

- Платформа: Windows x64

- Аудиоформаты (декодирование): через `soundfile` / `libsndfile`
  - WAV, AIFF/AIFC, FLAC, OGG/Vorbis, AU, RAW, MAT(MATLAB), PAF, SVX, NIST, IRCAM, VOC, W64, RF64 и др.

- Частоты дискретизации / битность:
  - 44.1 / 48 / 96 / 192 кГц
  - 16-bit / 24-bit / 32-bit float

- Внутренняя обработка:
  - non-destructive processing
  - float pipeline

- Вывод звука:
  - Стандартные аудиовыводы Windows (WASAPI / DirectSound / MME)
  - ASIO
  - Управление буфером/латентностью для ASIO — в планах
  - Ресемплинг — в планах

- Обработка звука (плагины): VST2
  - 32-bit
  - 64-bit (экспериментально)
  - поддержка цепочек эффектов; сохраняются все параметры плагинов
  - импорт/экспорт цепочек — в планах

- Waveform (форма сигнала):
  - построение волны аудиофайла с настраиваемыми параметрами
  - кэширование для максимальной скорости повторного открытия и отрисовки

- UX-функции (для аудиокниг):
  - закладки с сохранением позиции
  - управление из трея — в планах

- Архитектура:
  - модульная архитектура
  - шина сообщений между модулями (платформа для расширений)
