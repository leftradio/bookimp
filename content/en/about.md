## Description

> A compact, feature-rich PC player, created as part of the "SViD" project — Tales for Adults and Children — designed specifically for audiobooks.<br><br>Independent volume control for the narrator’s voice and the music/background track, allowing you to fine-tune (for yourself or for a specific recording) the “voice/background” balance.<br><br>VST2 plugins via a custom host: you can correct audio while listening — set up EQ, reduce excessive reverb (de-reverb) on problematic recordings, and use other processing chains, achieving the highest possible quality (including with professional plugins such as FabFilter Pro).

<br>

## Key features

- Platform: Windows x64
- Core features: play/pause, seeking, navigation, bookmarks
- Separate volume control: narrator voice / music background
- Online SViD book library
- VST2 (custom host): real-time correction while listening: EQ, de-reverb for problematic recordings, other processing chains, maximum audio quality when using high-quality plugins
- Styles: multiple styles supported + built-in style editor
